🔑 Gemini API Integration

Set up Google Gemini API for 8825 Meeting Automation

1 Get Your API Key

Instructions:

  1. Click "Open Google AI Studio" below
  2. Click "Create API Key" button
  3. In the dialog, you'll see:
    • Name your key: Enter "8825" (or any name)
    • Choose an imported project: Select "Default Gemini Project"
  4. Click "Create API key in new project"
  5. Copy the generated key (starts with "AIzaSy...")
Open Google AI Studio

💡 Tip: If you don't see "Default Gemini Project", you can select "Create project" to make a new one. The free tier includes 15 requests per minute.

2 Enter Your API Key

Your API key will be stored locally and never shared

3 Test & Save
⚠️ Requirements
  • Google AI Studio account (free tier available)
  • API key must start with "AIzaSy"
  • Key will be saved to: 8825_core/.env
  • Backend will automatically configure and activate
  • No restart required - works immediately
