OpenAI API Integration

Configure OpenAI GPT for 8825 Meeting Automation

1 Get Your API Key

You'll need an OpenAI account to get your API key.

Open OpenAI Platform
📋 Instructions:
  1. Click "+ Create new secret key" in OpenAI Platform
  2. Name your key: "8825"
  3. Set permissions: "All" (or customize as needed)
  4. Click "Create secret key"
  5. Copy the key (starts with sk-)
  6. ⚠️ Save it now - you won't see it again!
2 Enter Your API Key

Your API key will be stored locally and never shared

3 Test & Save
  • OpenAI account (free trial available)
  • API key must start with "sk-"
  • Key will be saved to: 8825_core/.env
  • Backend will automatically configure and activate
  • No restart required - works immediately
