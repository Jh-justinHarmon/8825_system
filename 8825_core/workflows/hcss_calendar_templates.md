## 🚀 QUICK CREATE WORKFLOW

### **Option 1: Google Calendar Web**
1. Open [Google Calendar](https://calendar.google.com)
2. Click "Create" button
3. Copy/paste template details
4. Select "harmon.justin@gmail.com" calendar
5. Set to recurring
6. Save

### **Option 2: Google Calendar Mobile**
1. Open Google Calendar app
2. Tap "+" button
3. Enter template details
4. Select "harmon.justin@gmail.com" calendar
5. Set to recurring
6. Save

### **Option 3: Bulk Create (Desktop)**
1. Open Google Calendar in browser
2. Create first event from template
3. Duplicate and adjust time for others
4. Faster than creating from scratch each time

**First-time setup:** 15-20 minutes (create all recurring events)

---

## 💡 TIPS

1. **Set all to recurring** - Only need to create once
2. **Use consistent naming** - Matches Microsoft calendar exactly
3. **Add "Microsoft Teams Meeting" to title** - Reminds you it's a Teams call
4. **Set 10-minute reminders** - Enough time to join
5. **Check times weekly** - Some meetings may shift
