## 🛠️ Available Tools

### HCSS MCP Tools (5)

1. **check_status**
   - Shows system status
   - Scheduler state
   - File counts
   - Last processing time

2. **ingest_gmail**
   - Triggers Gmail/Otter ingestion
   - Parameter: `days_back` (default: 7)
   - Returns: Processing summary

3. **list_recent_files**
   - Lists processed files
   - Parameters: `project`, `limit`
   - Returns: File list with metadata

4. **read_corrections_log**
   - Shows correction history
   - Parameter: `lines` (default: 50)
   - Returns: Recent corrections

5. **get_routing_stats**
   - Project distribution stats
   - Returns: Breakdown by project
