# Smart-HCSS Barter Partnership

**Status:** Active Pilot  
**Start Date:** 2025-11-08  
**Pilot Duration:** 90 days  
**Philosophy:** See `8825_core/philosophy/tokenization/`

---

## 🎯 OVERVIEW

Tokenized, cash-neutral barter between Smart Software and HCSS to exchange expertise:
- **Smart → HCSS:** RAL project engineering + IT management
- **HCSS (with 8825) → Smart:** Automation design + build (tools, MCPs, pipelines)

---

## 💰 TOKEN STRUCTURE

**Token Name:** 76C-Partner (non-transferable partner credit)  
**Peg:** 1 credit = $100 notional  
**Pilot Cap:** 600 credits per party (~$60,000 notional)  
**Expiry:** 18 months  

---

## 📊 SCOPE OF WORK

### **Smart → HCSS**
- Lightweight IT management (device mgmt, backup, identity, endpoints)
- RAL integration advisory
- SRE hygiene for HCSS's 8825 infra (monitoring, alerts, runbooks)

### **HCSS/8825 → Smart**
- Automation tools: intake hub, MCP routing, OCR/triage, job-aid generators
- RAL support tooling: structured workflow miners, bug triage dashboards
- Process bots: inbox → ticket creation, SLA trackers, change-log generators

---

## 🎓 PHILOSOPHY ALIGNMENT

This partnership is a **test case** for 8825's tokenization philosophy:

- ✅ **Value-based exchange** - Work measured by deliverables, not hours
- ✅ **Skill multipliers** - Scarce skills (1.25-1.5×), base work (1.0×)
- ✅ **Mutual benefit** - Both parties earn and spend
- ✅ **Governance** - Joint steering group (2 reps each, biweekly)
- ✅ **Expiration** - 18-month expiry encourages active use

**See:** `8825_core/philosophy/tokenization/principles.md`

---

## 📋 GOVERNANCE

**Steering Group:**
- **Smart:** 2 representatives
- **HCSS:** 2 representatives
- **Cadence:** Biweekly meetings
- **Responsibilities:** Approve scopes, set rate multipliers, issue/burn credits

**Quarterly True-Up:**
- If net balance >15% of cap → Roll over, cash out (80-100%), or apply to new work
