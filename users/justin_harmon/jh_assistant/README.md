## 🚀 Activation

### Enter Jh Mode
```
focus on jh
```

### Exit Jh Mode
```
exit focus
```

---

## 📋 Available Commands

### In Jh Mode:
- **List projects** - Show all active projects
- **Create project [name]** - Start new project
- **Open notes** - Access personal notes
- **Run script [name]** - Execute custom script
- **Archive [item]** - Move to archive

## 🎨 Workflows

### Personal Project Workflow
1. Create project folder
2. Set up README and structure
3. Work on project
4. Document progress
5. Archive when complete

### Note-Taking Workflow
1. Create note in notes/
2. Use markdown format
3. Tag and categorize
4. Link related notes

### Script Development
1. Create script in scripts/
2. Test and iterate
3. Document usage
4. Add to QUICKSTART
